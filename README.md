# Spam Email Detection using Machine Learning

## Objective
Classify messages as spam or not spam using NLP and Machine Learning.

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- NLP (TF-IDF)

## Algorithm Used
- Multinomial Naive Bayes

## Result
Model successfully detects spam messages with good accuracy.
